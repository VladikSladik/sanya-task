проект: Список персонажей из мультсериала "Рика и Морти" с использованием открытого API.

Приложение будет получать данные из публичного API, отображать их в виде списка карточек, и поддерживать базовые функции:

- Отображение списка персонажей;
- Поиск персонажей по имени;
- Фильтрация по статусу, виду и полу;
- Бесконечная загрузка списка (или ленивая загрузка);
- Редактирование некоторых данных о персонаже в его карточке (только локально, без постоянного сохранения);
- Просмотр детальной информации о персонаже на отдельной странице.
